# SDK Families

This directory contains SDK family workspaces, generation manifests, and generated SDK artifacts.

## Structure

- `sdkwork-music-app-sdk/`: Application SDK for user-facing features
- `sdkwork-music-backend-sdk/`: Backend SDK for administration
- `_route-manifests/`: Route manifest files
- `test/`: SDK verification tests

## SDK Generation

SDKs are generated from OpenAPI contracts using `@sdkwork/sdk-generator` / `sdkgen`.
