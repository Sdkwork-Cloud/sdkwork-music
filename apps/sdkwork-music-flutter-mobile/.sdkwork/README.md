# SDKWork Music Flutter Mobile Workspace

This directory contains local workspace metadata for the SDKWork Music Flutter Mobile application.

## Structure

- `skills/`: Local development skills and workflows
- `plugins/`: Application runtime plugins and extensions

## Purpose

This directory stores source-controlled workspace metadata, common skills, and local plugins for the Flutter mobile application surface. It is separate from generated SDK output and user-private runtime directories.
