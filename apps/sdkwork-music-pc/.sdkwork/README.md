# SDKWork Music PC Workspace

This directory contains local workspace metadata for the SDKWork Music PC application.

## Structure

- `skills/`: Local development skills and workflows
- `plugins/`: Application runtime plugins and extensions

## Purpose

This directory stores source-controlled workspace metadata, common skills, and local plugins for the PC application surface. It is separate from generated SDK output and user-private runtime directories.
