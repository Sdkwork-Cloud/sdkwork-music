# SDKWork Music H5 Workspace

This directory contains local workspace metadata for the SDKWork Music H5 application.

## Structure

- `skills/`: Local development skills and workflows
- `plugins/`: Application runtime plugins and extensions

## Purpose

This directory stores source-controlled workspace metadata, common skills, and local plugins for the H5 application surface. It is separate from generated SDK output and user-private runtime directories.
