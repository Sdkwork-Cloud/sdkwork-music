# sdkwork-music Skills

This directory holds application-local skills for `sdkwork-music`.

Add a skill only when the workflow is specific to this application and cites the
canonical SDKWork specs it follows, such as `API_SPEC.md`,
`SDK_SPEC.md`, `SDK_WORKSPACE_GENERATION_SPEC.md`,
`SDKWORK_WORKSPACE_SPEC.md`, `DATABASE_SPEC.md`, and `TEST_SPEC.md`.

Skills must not contain generated SDK output, runtime data, secrets, API keys,
auth tokens, local databases, or private user files.
